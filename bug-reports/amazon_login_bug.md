# Bug Report
Title: No erroe message displayed after wrong password

Steps to Reproduce:
1. Open amazon.de
2. Click on "Sign In"
3. Enter valid email
4. Enter wrong password
5. Click Login

Actual Result:
User is not logged in, but no error message is shown.

Expected Result:
User should see an error message indicating incorrect password.

Severety: Medium
Priority: Medium

Environment:
Browser: Chrome
OS: Windows 10
